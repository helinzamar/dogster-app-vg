# Dogster – Fakta om hundar

En enkel webbapplikation som visar slumpmässiga hundbilder med hjälp av [Dog CEO API](https://dog.ceo/dog-api/).

## Teknik

- **API:** Dog CEO API
- **Bibliotek:** Axios (API-anrop), SweetAlert2 (popup-rutor)
- **Byggverktyg:** Webpack

## Kom igång

```bash
npm install
npm run build
